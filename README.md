# ISC-S1- semestre 1 

##  APLs (Aplication programs)
son programs de aplicacion que ayudan al usuario a hacer tareas específicas como crear documentos, investigar y editar imágenes.

##  Asistente virtual 
son programs que usan la inteligencia virtual para ayudar al usuario a ralizar tareas con más fácilidad y eficiencia.

## Base de datos
Es un conjunto organizado de datos que se alamacenan de manera estructura para su acceso, gestión, y actualización .

## Backups ( copia de seguridad )
Es una copia de datos que se guarda para poder recuperarlos en caso estos se llegaran a perder o eliminar por error.

## CPU (unidad central de procesamiento)
Es el componente central de la computadora que se encraga de ejecutar instrucciones y procesar datos.

## Circuitos integrados 
Son dispositivos elctronicos pequeños que contiene multiples componentes ( transitores y resistencias) dentro de chips que pueden realizar funciones complejas.

## CI/CD ( integración continua/ entrega continua)
Es una practica en el desarrolo de un software que permite integrar cambios en los códigos de forma automática y frcuente, ademas de automátizar pruebas .

## Docker 
Es una plataforma que permite crear , ejecutar y distribuier aplicaiones dentro de contenedores, asegurando que funcionan igaual en cualquier entorno.

## DevOps 
Es una metodo que combina desarrolo (Dev) y operaciones (Ops) para mejorar colaboración, automatizar procesos y enteegar software de manera rápida y eficiente.

## Frontend
Es la parte del sitio o aplicacion con la que el usuario interractua directamente. eso incluye el diseño ,colores, textos, y todos los elemetos visibles en la pantalla.

## Git
Es un sistema de control o versiones que permite gestionar y registrar, los cambios en un codigo de un proyecto, facilitado el trabajo en equipo.

## Hardware 
es el conjunto de componentes de una computadora o un dispositivos que se pueden ver y tocar.

## Ley de moore
Es una observación realizada por Gordon Moore que establece que la cantidad de transistores en un chip se duplica aproximadamente cada dos años, aumentando la capacidad de procesamiento.

## Nube
Es un modelo de almacenamiento y servicios en internet que permite alamcenar datos y ejecutar aplicaciones remotamenten lugar de hacerlo en una computadora local.
